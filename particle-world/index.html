Description:

My project is a classic popcorn bucket with popcorn poping out of it. I wanted to create a fun particle drawing that everyone would know about it. What is Object-Oriented Programming (OOP), a Class and an Instance of the Class?

Object-Oriented Programming is what it sounds like. It is a code the focuses on the object of the code. It contains both the display of the function as well as the updates. A class is when similar data can be clumped together to form a "class". Then you can multiply that initial class. An instance of the class is when we used the word "new" followed by the name of the class.

Discuss the effectiveness of OOP. When is OOP useful? In what way can it be utilized?

OOP is very effective. It simplifies your code down and allows you to have each object to contain its own features and updates. It can be utilized to making particles and allow for greater reusability of the code.
